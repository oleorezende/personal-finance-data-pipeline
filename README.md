# 📊 Gasto do Cartão

Dashboard de controle financeiro pessoal desenvolvido a partir de dados reais exportados do Nubank.

## 🚀 Sobre o Projeto

Este projeto tem como objetivo transformar dados brutos de transações financeiras em insights visuais claros e úteis.

O processo envolve:
- Extração de dados via CSV do Nubank
- Tratamento e consolidação com Python (Pandas)
- Geração de base única em Excel
- Desenvolvimento de dashboard interativo em HTML

## 🧠 Tecnologias Utilizadas

- Python
- Pandas
- Excel
- HTML / CSS / JavaScript

## 🔄 Pipeline de Dados

1. Exportação dos dados do Nubank (CSV)
2. Limpeza e padronização dos dados com Pandas
3. União de múltiplos arquivos em uma única base
4. Criação de métricas e indicadores
5. Exportação para Excel
6. Construção do dashboard em HTML

## 📈 Funcionalidades do Dashboard

- Visão geral de gastos (bruto e líquido)
- Média mensal de despesas
- Evolução dos gastos ao longo do tempo
- Distribuição por categorias
- Top 5 maiores gastos
- Filtro por mês e categoria
- Busca por transações

## 📊 Principais Insights

- Identificação das categorias com maior impacto financeiro
- Análise de comportamento de consumo ao longo dos meses
- Controle de gastos recorrentes
- Visualização rápida de despesas mais relevantes

## 🎯 Objetivo

Este projeto foi desenvolvido com foco em:
- Prática de análise de dados
- Construção de dashboards
- Estruturação de pipeline de dados
- Criação de portfólio para área de dados
